</h2>
INGREDISAFE is a modern, personalized health assistant web application. It allows users to scan food nutrition labels and cosmetic ingredient lists via image upload or camera capture.
Leveraging Google Gemini, the app extracts ingredients, cross-references them with the user's specific health and skin profile, calculates personalized Acceptable Daily Intake (ADI) levels, flags drug-additive interactions, and provides an interactive AI Q&A chatbot for direct safety advice.
